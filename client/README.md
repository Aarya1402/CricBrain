# CricBrain AI 🏏🧠

**CricBrain AI** is a modern, cinematic cricket analytics platform built for the **Agentic Premier League (APL) Watch Party & Micro-Hackathon** organized by **GDG Baroda**.

Built using **"Vibe Coding"** principles, this project transforms complex IPL match statistics into engaging, AI-driven narratives and simplified insights for both casual fans and data enthusiasts.

## 🚀 Key Features

- **Agentic Match Dashboard**: Real-time visualization of match dynamics (KKR vs GT IPL Clash).
- **AI Insight Engine**: Deep-dive analysis into match momentum and strategic leverage points.
- **Explain Like I'm New**: Interactive "Beginner Mode" tooltips that simplify technical cricket jargon (CRR, RRR, Win Prob).
- **Momentum Timeline**: A high-impact visualization showing the ebb and flow of match momentum.
- **Cinematic Player Cards**: Dedicated impact profiles for star performers like Andre Russell and Rashid Khan.
- **AI Commentary Generator**: Auto-generated match summaries with a premium typewriter effect.

## 🛠️ Tech Stack (Vibe Coding Approved)

- **Frontend**: React.js + Vite
- **Styling**: Tailwind CSS v4 (Glassmorphism & Neon Aesthetics)
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React
- **Branding**: CricBrain Custom Identity

## 🎨 Design Philosophy

Inspired by premium sports broadcast overlays, CricBrain AI utilizes:
- **Glassmorphism**: Layered, translucent UI components for a modern feel.
- **Neon Accents**: High-contrast colors for crucial match insights.
- **Agentic UX**: Designed to behave intelligently, offering help exactly when the fan needs it.

## 📦 Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Start the Vibe: `npm run dev`.
