# CricBrain AI 🏏🧠

**CricBrain AI** is a modern, cinematic cricket analytics platform designed for the 2026 Hackathon. It transforms complex match statistics into engaging, AI-driven narratives and simplified insights for fans.

## 🚀 Features

- **Dynamic Match Dashboard**: Real-time visualization of match stats (KKR vs GT IPL Match).
- **AI Insight Engine**: Deep-dive analysis into match momentum and strategic shifts.
- **Explain Like I'm New**: Beginner-friendly tooltips for complex cricket terminology (CRR, RRR, Win Prob).
- **Momentum Timeline**: High-impact area charts showing the flow of the game.
- **Cinematic Player Cards**: Highlighting key performers with dynamic impact ratings.
- **AI Match Summary**: Auto-generated commentary summaries with a typewriter effect.

## 🛠️ Tech Stack

- **Frontend**: React.js + Vite
- **Styling**: Tailwind CSS v4 (with Glassmorphism & Custom Themes)
- **Animations**: Framer Motion
- **Visualizations**: Recharts
- **Icons**: Lucide React
- **Typography**: Inter / Outfit (Modern Sans)

## 🎨 Design Philosophy

Inspired by premium sports broadcasts, the UI features:
- **Dark Mode First**: Midnight backgrounds with neon accents.
- **Glassmorphism**: Layered, translucent cards for a depth-rich experience.
- **Micro-interactions**: Smooth transitions and hover effects for high engagement.

## 📦 Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Start the dev server with `npm run dev`.
